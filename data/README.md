# Data Folder

This folder contains datasets and annotations for the VQA-IR-QA project.

## Directory Structure

### `annotations/`
Contains all annotated QA pairs and label-studio annotation exports:
- `final_annotations/` - Processed and finalized QA annotations
  - `filtered_annotations/` - Filtered QA pairs by category and rephrasings
  - `merged_annotations/` - Merged annotation files from different sources
  - `rq_3/` - Annotations for research question 3

### `pages/`
PDF pages extracted and indexed for retrieval tasks.
